# Tiny3D
A Small OpenGL Based Renderer  

### Features:    

- OpenGL4.5 based  
- Instanced rendering  
- Post processing  
- .obj model file    
- .bmp texture file  
- Skeleton animation  
- Animation control  
- Vertex texture fetch  
- 3rd person view  
- Cascaded shadow map  
- Sky dome  
- Multi-thread support  
- Deferred shading  
- Dynamic Level of Detail  
- Billboard  
- Water  
- Fog  
- Depth of Field  
- Antialiasing  
- Screen Space Reflection  
- Screen Space Grass  
- PBR Material  
- GPU occlusion culling  
- Atmospheric scattering  
- Volumetric cloud  
  
### Control:  

- num-1,num-2,num-3 switch role to control  
- numpad 2486 to control light direction  
- v cancel role control    
- wasd move  
- space jump  
- mouse-left attack  
- mouse-right defend
